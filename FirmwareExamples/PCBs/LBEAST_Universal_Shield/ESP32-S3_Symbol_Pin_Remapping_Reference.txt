ESP32-S3 Symbol Pin Remapping Reference for 2x22 Header Footprint
====================================================================

This file provides the complete pin number mapping from the original ESP32-S3 WROOM symbol
to the custom 2x22 header footprint (44 pads).

INSTRUCTIONS:
1. Open your ESP32-S3 symbol in KiCAD Symbol Editor
2. For each pin listed below, change its Pin Number to the "New Pin #" value
3. Add the 3 duplicate pins listed at the end
4. Set unused pins to "No Connect" or delete them

PIN MAPPING TABLE (Original Symbol Pin → New Footprint Pad Number)
==================================================================

Power Pins:
-----------
VDD3P3 (original pin 2)          → New Pin #: 1
VDD3P3_2 (NEW - ADD THIS)         → New Pin #: 2  [Power Input - Duplicate for current dissipation]
VDD3P3_RTC (original pin 20)     → [Not used in footprint - set to No Connect]
VDD3P3_CPU (original pin 46)     → [Not used in footprint - set to No Connect]
VDD_SPI (original pin 29)         → [Not used in footprint - set to No Connect]
VIN/5V (if exists)                → New Pin #: 21

GND (original pin 57)             → New Pin #: 22
GND_2 (NEW - ADD THIS)             → New Pin #: 23  [Power Input - Duplicate for current dissipation]
GND_3 (NEW - ADD THIS)             → New Pin #: 43  [Power Input - Duplicate for current dissipation]
